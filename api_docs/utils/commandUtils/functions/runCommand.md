# Function: runCommand()

> **runCommand**(`command`, `workingDir`?): `string`

Runs a command in the specified directory.

## Parameters

• **command**: `string`

The command to run.

• **workingDir?**: `string`

The path to the working directory. If not provided, the current directory is used.

## Returns
